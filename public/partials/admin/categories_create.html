Create Custom Categories

Description

This will create custom Categories, Sub categories for end user. It will support up to 5 level

For Create >>

  • Step 1. click new for refresh
  • Step 2. write down title of your category(should be unique name)
  • Step 3. select categorylist from bottom by clicking checkboxes
  • Step 4. if you want to present it as subcategory, click checkbox of already defined category
  • Step 5. if you want to present it as main category, unselect custom category list check boxes if any exist/ or checked DEFAULT PARENT
  • Step 6. Click Create
    • of a new category select checkbox to define it's parent. and select prosperent categories for mapping

    For Edit >> click specific item's edit option . reorganize everything. If you want to redefine parent you can select new parent from tree

    • Step 1. click edit button beside the your category
    • Step 2. select check box of your desired parent
    • Step 3. Click Update
    if you want to make a category from child to main parent
    • Step 1. click edit button beside the category
    • Step 2. choose Default Parent and click

    For Delete >> click specific item's delete option .
